JZip
====

GUI ZIP Archive Manager for Korean


BUILD
-----

Create a symbolic link for swt.jar according to your machine architecture.

	ln -s swt/swt-linux-64.jar swt.jar

Make JZip.jar.

	scons


RUN
---

	java -jar JZip.jar
